# The Gallery

The Gallery adds a new button to the in-game Compendium screen.

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/8a458565-896b-44a3-a2cd-c1e773fba0c4" />

The button brings the user to a new screen with two tabs: a VFX browser and an audio browser. These make it easy for modders to reference VFX & audio paths for their own projects.
